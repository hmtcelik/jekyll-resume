Basic Blog Website Built with Python Django 3.2.9

### Features:
- Blogger login
- Blogger Register
- Adding new post after login (in website)
- Edit, delete current posts (in admin panel)
- Homepage,Aboutpage,Blogpage
- Basic footer and Basic navbar
